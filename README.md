# Golf Charity Subscription Platform

Deployable MERN implementation of the PRD from Digital Heroes. The app includes:

- Public marketing website with charity-led storytelling
- JWT auth with subscriber and admin roles
- Monthly and yearly subscription state management
- Stableford score tracking with automatic 5-score rolling retention
- Charity selection and contribution tracking
- Draw simulation and publishing workflow
- Admin dashboard for users, charities, winners, and analytics

## Stack

- Frontend: React + Vite + React Router
- Backend: Node.js + Express + MongoDB + Mongoose
- Auth: JWT
- Deployment target: Vercel (frontend) + Render/Railway (backend) + MongoDB Atlas
